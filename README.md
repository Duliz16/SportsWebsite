# 🏅 Sports E‑Commerce Website

## 🧑‍💻 Purpose

A fully‑functional multi‑page sports e‑commerce website built as part of the Web Design & Development module.
The project demonstrates HTML, CSS, JavaScript, DOM manipulation, form validation, sessionStorage, multi‑page navigation, and interactive UI components.

## 📌 Project Overview
This website showcases a complete sports‑themed platform with:

+ A timed presentation/landing page

+ A shared main homepage with consistent navigation

+ A full shop system (clothes, shoes, racquets, balls)

+ A cart system with session‑based storage

+ A checkout form with JavaScript validation

+ A dynamic invoice/summary generator

+ A sports content section with anchors

+ A page editor for each contributor

+ Additional interactive pages (quiz, thumbnails, sitemap, student details)

## 👤 My Role 

I was responsible for:

1. Presentation Page
   
+ Designed the introductory page with university logos and group details
  
+ Implemented a 4‑second auto‑redirect using JavaScript
  
+ Styled the page with a custom background and layout
  
+ Included a structured table of group members

2. Buy Products System (Shop Pages + Cart)
   
+ I built the entire shopping workflow:

3. Product Pages (Clothes, Shoes, Racquets, Balls)

+ Designed 4 separate product pages instead of one cluttered page

Each product includes:

+ Image

+ Title

+ Price

+ Quantity selector

+ Add to Cart button and Functionality
  
Implemented using sessionStorage + JSON

Products persist across pages

Handles multiple items and quantities

Validates quantity selection

4. Cart Page
   
Dynamically loads products from sessionStorage

Displays:

+ Product image, Name, Unit price, Quantity, Subtotal

+ Includes Remove functionality

+ Built a responsive grid layout for the cart

5. Checkout Form

Collects:

+ First name, Last name, Contact number, Email, Address

+ Includes Reset and Place Order buttons

+ JavaScript validation (no HTML5 validation allowed)

+ Alerts user if required fields are missing

6. Order Summary / Invoice

Displays:

+ Customer details, All selected products, Subtotals, Grand total

+ Uses DOM manipulation to generate the invoice dynamically

+ Clears cart after order is placed

7. Extra Content Page (Sports Section – Table Tennis)

+ Wrote and designed the Table Tennis section

+ Added images, headings, and structured content

+ Used anchor navigation for long‑form content

8. Page Editor

Created a personal editor page listing:

+ My name, Student ID, Role, All tasks completed

## 🛠️ Technologies Used
